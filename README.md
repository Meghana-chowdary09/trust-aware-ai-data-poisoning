# 🔐 Trust-Aware AI for Data Poisoning Detection

## 📌 Overview
This project presents a Trust-Aware AI system designed to detect and mitigate data poisoning attacks in machine learning pipelines.
It combines unsupervised anomaly detection using Autoencoders with supervised classification using XGBoost, enhanced by a dynamic 
trust scoring mechanism. The system improves model robustness by identifying malicious data patterns, adapting to evolving threats,
and maintaining reliable performance in adversarial environments.
---

## 🚀 Features
- Anomaly detection using Autoencoder  
- Supervised classification using XGBoost  
- Trust-aware learning mechanism  
- Data preprocessing and pipeline automation  
- Model evaluation and monitoring
